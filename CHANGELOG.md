# Changelog

All notable changes to this project will be documented in this file.

## [4.8.1] - 2026-02-25
### Added
- 手动备份与恢复功能（本地覆盖式备份）。
- 会话级别冲突检测标志：首次登录执行冲突检测，减少重复提示。
- 在认证下拉菜单中显示手动备份信息；将保存/恢复作为菜单项。
- UI 小修：默认主题设为浅色，手动备份高亮色调整为橙色，移除账户下的邮箱显示（部分 GitHub 账号不提供邮箱）。

### Fixed
- 修复多处同步与冲突提示的 UX 问题。

## [4.8.0] - prior
- 初始发布，前后端分离、GitHub OAuth、云端同步、离线优先等功能。
