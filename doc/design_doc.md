# 《烟雨江湖》装备录入工具设计文档 (V4.7.30)

## 1. 项目概述
本工具是一个基于 Web 的单页面应用（SPA），旨在帮助《烟雨江湖》玩家记录和对比不同地点（如剑王阁、铸剑山庄等）的装备锻造/缝纫序列，以便优化资源分配。

## 2. 系统架构
- **前端技术栈**: 原生 HTML5, CSS3 (使用变量实现主题切换), Vanilla JavaScript (无第三方库依赖)。
- **持久化方案**: 使用浏览器 `localStorage` 存储用户数据（键名为 `yanyu_v24_db`）。
- **部署方式**: 纯静态页面，可直接离线运行。

## 3. 核心功能模块

### 3.1 导航与统计 (Nav Area)
- **分类切换**: 支持“铸造”（武器、戒指、项链）和“缝纫”（护甲、护腕、腰带）两大系统的切换。
- **实时统计**: 按钮上方显示已录入的序列数量（Badge）；右侧显示各地点（如剑王阁、塞北）的具体录入统计。

### 3.2 录入控制 (Control Head)
- **起始位设置**: 用户可以自定义每个装备的起始录入次数，方便对齐游戏内的实际进度。
- **快速录入**: 提供“绿”、“蓝”、“紫”三个质量等级按钮，点击打开属性选择器。

### 3.3 属性选择器 (Attribute Picker - Modal)
- **属性池管理**: 根据装备类型（武器/防具/饰品）自动加载对应的属性池。
- **特殊属性**: 集成了特定地点（如剑王阁命中、塞北防御）的特殊词条。
- **智能提交**: 当选择的词条数量达到品质上限（绿1/蓝2/紫3）且选择了特殊属性后，支持自动提交。

### 3.4 序列展示与操作 (Main Grid)
- **双侧对比**: 同一界面展示该地点的两个对应装备序列，方便对比（例如同时看拳套和刀）。
- **操作项**:
    - **修改 (✎)**: 重新打开选择器编辑已有条目，支持跨列调整。
    - **收藏 (★)**: 标记优秀序列，整行高亮显示。
    - **删除 (✕)**: 移除特定位置的录入记录。

### 3.5 数据维护 (Control Panel)
- **主题切换**: 支持深色/浅色模式。
- **备份/还原**: 通过 JSON 文件导出和导入 `localStorage` 数据，防止数据丢失。
- **清空**: 一键重置所有数据。

## 4. 数据结构设计
存储在 `localStorage` 中的 JSON 对象结构如下：
```json
{
  "装备类型 (如: 拳套)": {
    "地点 (如: 剑王阁)": {
      "装备名 (如: 破军指套)": [
        {
          "q": "purple",    // 品质 (green/blue/purple)
          "attrs": ["攻击", "命中"], // 基础属性数组
          "special": "臂力+5", // 特殊属性
          "k": true        // 是否收藏 (keep)
        },
        null, // 表示该位置未录入
        ...
      ]
    }
  }
}
```

## 5. 关键逻辑说明
- **滚动锁定**: 录入或编辑后，页面会自动滚动到对应的行（`focusScroll` 逻辑），提升连续录入的体验。
- **兼容性逻辑**: `fixOldData` 函数负责处理旧版本数据中的词条名称差异（如“剑伤”转换为“剑法”）。
- **自适应布局**: 使用 CSS Grid (`loc-grid`) 实现响应式布局，保证在不同分辨率下的对比体验。
